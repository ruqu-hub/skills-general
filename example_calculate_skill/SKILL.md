# 计算器示例 Skill

一个简单的 Python 计算器脚本工具。

## 适用场景

✅ **在以下场景使用：**
- 需要进行基础的加、减、乘、除运算。

## 使用指南

直接运行 `example_calculate_skill.py` 脚本：

### 算术运算
```bash
# 加法
python example_calculate_skill.py add 10 5

# 减法
python example_calculate_skill.py subtract 10 5

# 乘法
python example_calculate_skill.py multiply 12 8

# 除法
python example_calculate_skill.py divide 100 4
```

### 配置
如需设置 API Token，请使用：
```bash
python example_calculate_skill.py set-token <your_token>
```
