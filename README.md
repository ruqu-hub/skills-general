# skills-general

面向 AI Agent 的通用技能库 (General skills for AI agents)。

## 包含的技能

### [example_calculate_skill](./example_calculate_skill/SKILL.md)
一个简单的 Python 计算器工具，支持基础的加、减、乘、除运算。

- **功能**: 执行加 (add)、减 (subtract)、乘 (multiply)、除 (divide) 运算。
- **环境要求**: Python 3.x。

## AI 安装与使用指南

如果你是一个 AI Agent，请按照以下步骤安装并使用本仓库中的技能：

### 1. 克隆仓库
首先，将本仓库克隆到你的工作区的 `ruqu-hub/skills-general` 目录下。

### 2. 拷贝技能到你的技能目录
选择你需要的技能文件夹，将其拷贝到你的 AI 技能存放目录

### 3. 配置建议
- 确保你的环境中已安装 Python。
- 对于需要 API Token 的技能，参考各技能目录下的 `SKILL.md` 进行配置。
